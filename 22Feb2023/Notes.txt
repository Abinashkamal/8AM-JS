Non-Primitive Data Types:

Arrays :

An array is a linear data structure which stores the data in the
sequential order. 

in JS, Array can store any type of data.

syntax:

 var array_name = [];

  array_name should be a noun. 

Array index always starts with 0 and using index values we can access the 
actual data. 


code:

  var array = [10, 20, true, false, "hello"];
        console.log(array);
        console.log(array[0]);
        console.log(array[3]);
        console.log(array[5]);
        console.log(array);


methods/ functions:

 var array = [10, 20, true, false, "hello"];
        array.push(100);
        array.unshift(0);
        console.log(array);
        array.pop();
        array.shift();
        console.log(array);
        var index = array.indexOf(false);
        console.log(index);
        array.splice(index, 1);
        console.log(array);
        var index2 = array.indexOf(20);
        console.log(index2);
        array.splice(index2, 0, 5);
        console.log(array);
